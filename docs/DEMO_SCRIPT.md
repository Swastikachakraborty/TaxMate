## Tips for Recording

1. **Practice the flow 3x** before recording — smooth navigation is key
2. **Use the demo personas** (Ravi) — pre-seeded data ensures everything works
3. **Zoom in** on key numbers when narrating specific amounts
4. **Keep cursor movement smooth** — avoid jittery mouse movements
5. **Background music**: Subtle lo-fi or corporate tech music (low volume)
6. **Resolution**: Record at 1080p minimum, 1440p preferred
